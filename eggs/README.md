# Sodium Panel Eggs

Sodium is compatible with Pterodactyl eggs. Import any `.json` egg file through Admin > Eggs.

## Adding More Eggs

Download eggs from:
- https://github.com/pterodactyl/game-eggs
- https://github.com/pelican-eggs

Then import via Admin > Eggs > Import Egg.
